# Examen Mi-Session – Programmation Web Front-End

## 📌 Description

Ce projet a été réalisé le 10/09/2025 dans le cadre de l’examen de mi-session du cours de programmation web front-end.

L’objectif est de concevoir un site web simple en utilisant **HTML et CSS**, avec une structure claire et une navigation entre plusieurs pages.

---

## 🎯 Objectifs

* Créer une structure HTML valide
* Utiliser CSS pour le design et la mise en page
* Organiser les fichiers correctement (images, pages, styles)
* Créer une navigation entre les pages
* Respecter les consignes de l’examen

---

## 🛠️ Technologies utilisées

* HTML5
* CSS3

---

---

## 🚀 Fonctionnalités

* Page d’accueil (`index.html`)
* Navigation entre les pages
* Page services
* Page contact
* Intégration d’images
* Mise en page stylisée avec CSS

---

## ▶️ Comment exécuter le projet

1. Télécharger ou cloner le projet
2. Ouvrir le dossier dans Visual Studio Code
3. Ouvrir le fichier `index.html`
4. Lancer avec **Live Server** (recommandé)
